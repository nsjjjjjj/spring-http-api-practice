1. 프로젝트 정보

언어: Java 100%

빌드 도구: Maven

주요 구성: Maven Wrapper, src(소스 코드), pom.xml

2. 프로젝트 구조

src/main/java: 실제 비즈니스 로직과 알고리즘이 구현된 소스 코드

src/test/java: 코드의 정상 작동 여부를 확인하기 위한 테스트 케이스

pom.xml: 프로젝트에 필요한 라이브러리 및 빌드 옵션 설정 파일

3. 주요 학습 내용

Maven을 활용한 프로젝트 빌드 및 라이브러리 관리 프로세스 익히기

2주차 과정에서 다룬 주요 Java 개념(예: 예외 처리, 컬렉션 프레임워크, 파일 I/O 등) 실무 적용

일관된 프로젝트 구조 유지를 통한 협업 기초 다지기
