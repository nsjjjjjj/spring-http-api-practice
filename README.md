1. 프로젝트 정보

성격: 주차별 학습 과제 및 실습 프로젝트

주요 기술: Java 17, Maven

주요 테마: 프로젝트 구조화, 인터페이스 설계, 컬렉션 프레임워크 실전 적용

2. 학습 및 구현 내용

Maven을 활용한 프로젝트 관리: pom.xml을 통한 의존성 관리와 빌드 생명주기(Lifecycle)에 대한 이해를 바탕으로 프로젝트를 구성했습니다.

인터페이스와 다형성: 인터페이스를 활용하여 유연한 구조를 설계하고, 다형성을 통해 코드의 재사용성을 높이는 연습을 진행했습니다.

자바 컬렉션(Collections) 심화: 다양한 자료구조(ArrayList, HashMap 등)를 상황에 맞게 선택하여 데이터를 효율적으로 처리하는 로직을 구현했습니다.

예외 처리 전략: 프로그램의 안정성을 위해 try-catch와 사용자 정의 예외를 활용한 예외 처리 로직을 설계했습니다.

3. 기술 스택 (Tech Stack)

Language: Java

Build Tool: Maven

IDE: IntelliJ IDEA

Version Control: Git

4. 실행 및 빌드 방법
본 프로젝트는 Maven Wrapper를 포함하고 있습니다.

저장소 클론: git clone https://github.com/nsjjjjjj/secondWeekProject.git

프로젝트 빌드: ./mvnw clean compile (Windows는 mvnw.cmd)

테스트 실행: ./mvnw test

5. 주차별 회고

1주차에서 배운 기초 문법을 바탕으로, 2주차에는 실제 '프로젝트'다운 구조를 갖추는 과정을 경험했습니다.

특히 Maven을 사용하여 라이브러리를 관리하고 빌드하는 표준적인 Java 개발 환경에 익숙해지는 계기가 되었습니다.
